# cmake-gtest
This is an example project of using [cmake](https://cmake.org) as its build tool and [gtest](https://github.com/google/googletest) as its unit test tool.

Build like this:
```
mkdir build
cd build
cmake ..
make
```

To run main program (inside `build/`):
```
make run
```

To test (inside `build/`):
```
make test
```
